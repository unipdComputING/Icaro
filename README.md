**ICARO**

Code developed with femDEV 2024-2025 team (UNIPD)

2D FEM, HEAT TRANSFER SOLVER (Steady state and Transient)

**Features:** 
- generate mesh with parametric settings for armored concrete sections
- import mesh
- multi-material propriety
- assign different boundary conditions
- export compatible with ParaView (.vtk)

**Future development:**
- time-dependent load amplitude (fire curve)
